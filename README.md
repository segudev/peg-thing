# Peg thing

A terminal implementation of Peg Thing written in Clojure

## Installation

You need Leiningen to run the game

https://leiningen.org/#install

## Usage

    cd ./peg-thing
    lein run

## Example

```
Get ready to play peg thing!
How many rows? [5]
5
Here's your board:
      a0
     b0 c0
   d0 e0 f0
  g0 h0 i0 j0
k0 l0 m0 n0 o0
Remove which peg? [e]
```
